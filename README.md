# 公考小工具

面向公务员考试备考的资料处理工具合集。基于 PyQt6 桌面应用，提供 PDF 格式转换、内容清洗、标准排版等功能。工具采用统一的插件式架构，可按需扩展。

---

## 🧰 工具列表

| 工具 | 说明 | 使用文档 |
|------|------|:--:|
| PDF 格式转换 | 将粉笔/华图/中公等刷题软件导出的 PDF 转换为国考标准格式，支持行测和申论 | [📖 使用说明](./tools/pdf_converter/docs/pdf-converter.md) |
| 考试计时器 | 正计时 / 倒计时，大字显示，支持分段记录 | [📖 使用说明](./tools/exam_timer/docs/exam-timer.md) |

## 🏗 项目架构

项目采用 **UI 层 → 配置层 → 核心处理层** 的三层架构，各工具共享同一套基础设施。

```
CivilServantsTools/
├── main.py                          # 应用入口
├── setup.bat / run.bat              # 一键安装 / 启动
├── requirements.txt                 # Python 依赖
│
├── ui/                              # 共享 UI 框架
│   └── main_window.py               # 主窗口（首页 + 工具页路由）
│
├── tools/                           # ★ 功能模块目录
│   ├── pdf_converter/               # PDF 格式转换模块
│   │   ├── config/                  #   配置（模板 / 过滤规则）
│   │   ├── core/                    #   核心处理（解析 / 清洗 / 排版 / 生成）
│   │   ├── ui/                      #   工具界面（预览 / 设置 / 工具栏 / 工作线程）
│   │   └── docs/                    #   使用文档
│   └── exam_timer/                  # 考试计时器模块
│       ├── core/                    #   计时引擎 + 数据模型
│       ├── ui/                      #   计时器界面
│       └── docs/                    #   使用文档
│
├── resources/
│   └── styles/app.qss               # Qt 全局样式表
│
├── output/                          # 调试输出目录
└── sample_input/                    # 示例文件
```

### 架构设计

```
┌─────────────────────────────────────────┐
│                UI 层 (PyQt6)             │
│  首页 → 工具卡片 → 工具页（预览+设置）    │
├─────────────────────────────────────────┤
│              配置层 (JSON)               │
│  排版模板 / 过滤规则 / 模块识别规则       │
├─────────────────────────────────────────┤
│            核心处理层 (Pipeline)          │
│                                          │
│  ① 解析 ──→ ② 清洗 ──→ ③ 排版 ──→ ④ 生成 │
│  PyMuPDF    xingce    Layout     ReportLab│
│  PaddleOCR  shenlun   Engine     FontMgr │
└─────────────────────────────────────────┘
```

- **UI 层**：PyQt6 桌面应用，`main_window.py` 为共享 Shell，首页展示工具卡片，点击进入对应工具页面。各工具的界面和逻辑独立存放在 `tools/<tool_name>/ui/` 下。
- **配置层**：JSON 模板驱动，排版参数（字体/字号/行距/边距等 40+ 项）、过滤规则、模块识别规则均从配置文件加载，无需改代码即可调整。
- **核心层**：每个工具的 `core/` 独立管理其处理逻辑。PDF 转换模块采用 解析→清洗→排版→生成 四阶段流水线，`models.py` 定义 13 个 dataclass 贯穿全流程，各阶段可独立替换或扩展。

## 🚀 快速开始

### 环境要求

- Windows 10/11（64位）
- Python 3.10 或更高版本
- 约 2GB 磁盘空间（含 OCR 模型约 500MB）

### 安装

```bash
pip install -r requirements.txt
```

或双击 `setup.bat` 自动检测 Python 环境并安装。

### 启动

```bash
python main.py
```

或双击 `run.bat`。

## 🛠 技术栈

| 组件 | 技术 | 用途 |
|------|------|------|
| UI 框架 | PyQt6 ≥ 6.6 | 桌面界面、信号/槽、双栏预览 |
| PDF 文本提取 | PyMuPDF ≥ 1.24 | 文本块提取、图片提取、预览渲染 |
| OCR 引擎 | PaddleOCR ≥ 2.8 | 扫描件中文识别 |
| PDF 生成 | ReportLab ≥ 4.2 | 矢量输出、中文字体嵌入 |
| 图像处理 | Pillow ≥ 10.0 | OCR 预处理、格式转换 |
| 数据验证 | Pydantic ≥ 2.0 | 配置模型验证 |

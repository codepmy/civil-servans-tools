# PDF格式转换器 — 公务员考试真题格式

将公务员考试刷题软件（粉笔、华图、中公等）导出的PDF，转换为国家考试标准格式（字体/行距/题号/选项排列）。

## ✨ 功能

- 📄 **PDF解析**: 自动识别文本型/扫描型PDF
- 🧹 **智能清洗**: 自动去除水印、广告、页码等非题目内容
- 📐 **标准排版**: 宋体正文、黑体题号、标准行距、A4页面
- 🎨 **实时预览**: 左右对比原始PDF和转换后的效果
- ⚙ **可调参数**: 字体、字号、行距、边距、选项排列等
- 📝 **双题型**: 支持行测（行政职业能力测验）和申论

## 🚀 快速开始

### 1. 环境要求

- Windows 10/11（64位）
- Python 3.10 或更高版本
- 约 2GB 磁盘空间（含OCR模型）

### 2. 安装

双击运行 `setup.bat`，或手动执行:

```bash
pip install -r requirements.txt
```

### 3. 启动

双击 `run.bat`，或:

```bash
python main.py
```

## 📖 使用说明

1. 点击 **📂 打开PDF** 选择刷题软件导出的PDF文件
2. 左侧预览区显示原始PDF
3. 在右侧设置面板调整排版参数:
   - **考试类型**: 行测 / 申论
   - **字体**: 标题/正文/题号/选项
   - **字号**: 各元素大小
   - **行距倍数**: 通常1.5倍
   - **选项排列**: 竖排 / 双栏
   - **页边距**: 上下左右
4. 点击 **▶ 开始转换**（或设置面板的"应用并转换"按钮）
5. 右侧预览区显示转换结果
6. 点击 **💾 保存PDF** 导出最终文件

## 🛠 技术栈

| 组件 | 库 |
|------|-----|
| UI框架 | PyQt6 |
| PDF文本提取 | PyMuPDF (fitz) |
| OCR(扫描件) | PaddleOCR |
| PDF生成 | ReportLab |
| 数据验证 | Pydantic |

## 📋 支持格式对照

| 刷题软件格式 | → | 考试标准格式 |
|------------|---|------------|
| 微软雅黑/黑体正文 | → | 宋体(SimSun) 10.5pt |
| 题号 "1." "1、" | → | "1．" 全角点号 |
| 选项 "A." "A)" | → | "A．" 全角点号 |
| 非标准行距 | → | 1.5倍行距 |
| 水印/广告/二维码 | → | 自动移除 |
| 无标准页眉 | → | "行政职业能力测验"/"申论" |

## ⚠ 注意事项

- OCR功能需要额外安装: `pip install paddleocr paddlepaddle`（首次使用需下载约500MB模型）
- 扫描型PDF的OCR处理速度约为2-5秒/页
- 输出PDF使用字体子集嵌入，可在任意设备正常显示中文

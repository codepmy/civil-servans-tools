## 🚀 四大核心功能

### 1. 📄 PDF 真题排版转换器（目前行测资料分析部分还存在一些缺陷）

将**粉笔 / 华图 / 中公**等 App 导出的 PDF 试卷，一键转换为**标准真题排版格式**。

`注：由于粉笔是使用Skia/Chromium PDF引擎生成，在文字提取上，可能会出发性能问题，导致提取较慢`

- **双引擎解析**：自动识别 PDF 类型，文字型走 PyMuPDF 文本解析，扫描型走 PaddleOCR 图文识别
- **智能清洗**：
    - 行测卷：自动去除水印、广告、二维码；识别题号与选项（A/B/C/D）；按五大模块归类（政治理论/常识判断、言语理解与表达、数量关系、判断推理、资料分析）；过滤页码等干扰信息
    - 申论卷：提取给定材料段落；分离作答要求；识别双栏排版；跳过答题卡页面
- **专业排版**：正文宋体（SimSun），题号黑体（SimHei），标准行距，字号 / 页边距 / 纸张大小均可配置
- **两套快捷预设**：国考（仿宋 14pt B5）和联考（仿宋 12pt B5），一键切换
- **图表保留**：资料分析模块的图表自动提取并嵌入
- **实时双栏预览**：左侧原始 PDF，右侧转换结果，所见即所得
- **可配置 40+ 排版参数**，JSON 驱动，无需改代码即可调优

### 2. ⏱️ 考试计时器

模拟真实考场计时环境的轻量计时工具。

- **正计时 / 倒计时**双模式
- 大字号显示（80px 加粗数字），远处也看得清
- **分段记录**（Lap）：点击记录各阶段耗时，有序列表展示
- 倒计时预警：剩余 5 分钟数字变红，归零时闪烁提醒
- 基于 Qt QTimer 引擎，100ms 精度

### 3. 📝 申论答题卡生成器

生成标准申论答题卡，支持打印练习或导出图片。

- **标准模式**：指定页数，每页独立编号，含 100/200/300/400/500/600 字数标记（25 列 × 24 行，每页 600 格）
- **题目模式**：按题目字数要求自定义，自动分页、跨页连续排版、题间距可设
- 导出为 **PDF**（ReportLab）或 **PNG 图片**（每页一张，PyMuPDF 渲染）

### 4. 📎 PDF 文件拼合

将多份 PDF 按自定义顺序合并为一份。

- **拖拽添加**：直接拖入 PDF 文件到列表，支持批量选择
- **拖拽排序**：列表内直接拖动调整合并顺序，半透明预览拖动目标
- **实时预览**：选中文件即时预览全部页面，支持翻页浏览
- **一键合并**：后台执行，进度可见，支持取消
- **统一保存**：默认文件名 `拼合后.pdf`

## 📥 安装与运行

### 方式一：从源码运行

```bash
# 1. 克隆仓库
git clone https://github.com/codepmy/civil-servans-tools.git
cd CivilServantsTools

# 2. 一键安装环境
setup.bat

# 3. 启动程序
run.bat
```

### 方式二：直接使用打包版

下载 `公考小工具.exe`，双击运行（已包含所有依赖和资源）。

> **注意**：OCR 功能需要 PaddlePaddle 运行环境，打包版 exe 已内置。首次启动 OCR 解析时可能需额外 1-2 分钟初始化模型。

## ⚠️ 已知限制

- 目前仅支持 **Windows 10 / 11（64 位）** 平台，macOS / Linux 尚未适配
- OCR 引擎（PaddleOCR）初始化较慢，首次使用需等待模型下载与加载
- 排版转换对**非标准版式**的第三方 App 导出 PDF 可能存在兼容性问题，建议优先使用粉笔 App 导出的 PDF
- 打包版 exe 体积较大（含 PaddlePaddle 运行库），约 140MB+

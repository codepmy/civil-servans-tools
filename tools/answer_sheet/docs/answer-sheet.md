# 申论答题纸生成器

用于生成可打印的申论答题纸，支持标准按页生成和按题目字数生成。

## 标准模式

- 每页固定 600 字，布局为 25 格 x 24 行。
- 用户只需设置页数。
- 每页独立从 1 开始计数，在 100、200、300、400、500、600 字所在格子的右下角添加标注。

## 分题模式

- 可添加或删除题目，每题独立设置目标字数。
- 题与题之间保留小间隔，连续排版，不强制新页。
- 遇到当前页剩余空间不足时自动分页，同一道题可跨页续写。
- 每页标注按本页从 100 开始重新计数；不足 600 字的末页只显示剩余格子，并标注该页内已有的整百位置。

## 导出

- 支持导出 PDF，适合直接打印。
- 支持导出 PNG 图片，每页输出一张图片。

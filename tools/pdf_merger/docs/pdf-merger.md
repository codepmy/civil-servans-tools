# PDF 文件拼合

将多份 PDF 按自定义顺序合并为一个文件，支持拖拽添加、拖拽排序、实时预览。

---

## 功能

- 📂 **拖拽添加**：将 PDF 文件直接拖入列表，也可通过"添加文件"按钮批量选择
- 🔀 **拖拽排序**：在列表中直接拖动调整合并顺序，拖动时半透明显示当前文件内容
- 📄 **实时预览**：点击列表中的文件即可预览，支持翻页浏览全部页面
- 🚀 **一键合并**：后台执行拼合，进度条实时反馈，支持中途取消
- 💾 **一键保存**：合并完成后点击"保存结果"，默认文件名 `拼合后.pdf`

---

## 使用说明

1. 在首页点击 **「📎 PDF文件拼合」** 进入
2. **添加文件**：将 PDF 文件拖入左侧列表，或点击 **「＋ 添加文件」** 按钮选择
3. **调整顺序**：直接在列表中上下拖动文件到目标位置，也可选中后点击 **「▲ 上移」「▼ 下移」**
4. **预览检查**：点击列表中的文件，右侧会显示该文件内容，可翻页确认
5. 点击 **「▶ 开始拼合」**，等待进度条完成
6. 拼合成功后弹出提示，点击 **「💾 保存结果」** 选择保存位置
7. 如需删除某个文件，选中后点击 **「✕ 移除」**（或按 Delete 键）

---

## 操作提示

- 列表中每个文件会显示**序号、文件名、页数和文件大小**，合并后按序号顺序排列
- 列表底栏实时显示**文件总数**和**预计合并后总页数**
- 拼合过程中可点击进度弹窗的"取消"按钮中止操作
- 拼合成功后：
  - 弹出绿色静默通知（无系统提示音）
  - 底栏状态变为绿色加粗确认信息
  - 保存按钮自动高亮为蓝色，引导你立即保存
- 重新调整文件列表后，之前的拼合结果会失效，需要重新执行拼合

---

## 注意事项

- 拼合后的 PDF **保留原始排版和内容**（图表、字体、格式等不变），使用 PyMuPDF 引擎确保无损合并
- 支持任意数量的 PDF 文件，但文件总页数过大时合并耗时可能较长
- 支持自定义页码范围：每个文件条目下方可设置"拼合 [起始] — [结束] 页"，默认合并全部页面
- 调整页码范围后，底栏预计页数会实时更新

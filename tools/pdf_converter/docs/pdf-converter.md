# PDF 格式转换

将公务员考试刷题软件（粉笔、华图、中公等）导出的 PDF，转换为国家考试标准格式。

---

## 功能特性

- 📄 **智能解析**: 自动识别文本型 / 扫描型 PDF，文本提取 + OCR 双引擎
- 🧹 **内容清洗**: 自动去除水印、广告、二维码、页码、页眉页脚
- 📐 **标准排版**: 宋体正文、黑体题号、标准行距，完美还原真题格式
- 🎨 **实时预览**: 左右对比原始 PDF 和转换效果，支持翻页
- ⚙ **灵活调节**: 字体、字号、行距、边距、纸张大小等全部可调
- 📝 **双题型**: 行测（五大模块）+ 申论（给定资料 + 作答要求）
- 🖼 **图表保留**: 资料分析中的表格/图表自动提取并嵌入
- 🚀 **一键预设**: 国考真题 / 联考真题字体快速切换

---

## 使用说明

1. 打开 PDF 文件（`Ctrl+O` 或工具栏"打开PDF"）
2. 左侧预览原始 PDF，翻页查看内容
3. 右侧设置面板调整参数：
   - **考试类型**: 行测 / 申论
   - **快速预设**: 国考真题（仿宋四号 B5）/ 联考真题（仿宋小四号 B5）
   - **纸张大小**: A4 / B5 / A3 / B4 / 16开
   - **字体**: 正文 / 题号 / 选项 / 页眉独立设置
   - **字号**: 各元素大小独立调节
   - **行距倍数**: 1.0 ~ 3.0 倍
   - **题间间距 / 选项缩进**
   - **页边距**: 上 / 下 / 左 / 右
   - **显示选项**: 答案横线 / 页码
4. 点击 **「▶ 应用并转换」** 或工具栏 **「开始转换」**
5. 右侧显示转换结果，对比确认后 **「保存PDF」**（`Ctrl+S`）

## 转换流水线

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  ① 解析   │ → │  ② 清洗   │ → │  ③ 排版   │ → │  ④ 生成   │
│  Parse    │    │  Clean    │    │  Layout   │    │ Generate  │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
     ↓               ↓               ↓               ↓
 ParsedDocument  CleanedDocument  LaidOutDocument   PDF Bytes
```

### ① 解析 — 从 PDF 提取结构化文本

**文本型 PDF**（`TextParser`）：PyMuPDF 逐行提取 text spans，保留字体名、字号、加粗属性和毫米坐标。同时通过 image xref、drawings、visual region erasure 三种方式提取图片块。

**扫描型 PDF**（`OCRParser`）：将页面渲染为 200 DPI 图片，EasyOCR 中文识别，CUDA 可用时通过 PyTorch 使用 NVIDIA GPU，置信度 ≥ 0.5 的文本保留。

自动判断 PDF 类型：平均每页字符 > 20 为文本型，否则回退 OCR。

页眉（< 28mm）和页脚（> 270mm）区域自动过滤，但边界附近的题目内容会被保留。

### ② 清洗 — 提取结构化题目

**行测**（`XingceCleaner`）:

| 步骤 | 说明                                    |
|------|---------------------------------------|
| 题号识别 | 正则 `数字 + .／．/、`，智能跳号检测（±3 容差）         |
| 选项提取 | A/B/C/D 四选项，支持跨行续接                    |
| 模块归类 | 政治理论+常识判断 / 言语理解 / 数量关系 / 判断推理 / 资料分析 |
| 广告过滤 | 关键字 + 位置判定（页角小字）                      |
| 图片区域保护 | 图表区文本不误提取                             |
| 页码过滤 | `第X页` / `共X页` / 纯数字短文本                |

**申论**（`ShenlunCleaner`）:

| 步骤 | 说明 |
|------|------|
| 给定资料 | 提取材料段落，保留首行缩进 |
| 作答要求 | 识别"第一题"/"1."两种编号，分离要求条目 |
| 作答纸跳过 | 自动识别方格纸/答题纸页面 |
| 双栏识别 | 作答要求页左右双栏分栏阅读 |
| 噪声过滤 | 水印 / 页码 / 空引号行 |

### ③ 排版 — 计算页面元素坐标

- 逐字符计算 ReportLab 文本宽度，精确断行
- 当前页不足时自动分页，页码自增
- 题号 + 题干首行同排，后续行对齐
- 选项缩进 + 续行悬挂缩进
- 资料分析：图表标题 → 图片 → 注释 → 题干 有序排版
- 申论：段落缩进 + 题号 + 要求条目悬挂缩进

### ④ 生成 — 渲染最终 PDF

- ReportLab 渲染，中文字体通过 FontManager 注册并嵌入
- 图片 PNG 嵌入，保持宽高比
- 可选答案横线
- 页码居中 `- N -` 格式

## 数据模型

| 模型 | 阶段 | 说明 |
|------|------|------|
| `TextBlock` | 解析 | PDF 文本块（文本 + mm 坐标 + 字体 + 字号） |
| `ImageBlock` | 解析 | PDF 图片块（二进制 + 坐标 + 尺寸） |
| `ParsedPage` / `ParsedDocument` | 解析输出 | 页面 / 文档 |
| `Question` / `Option` | 清洗 | 行测题目（题干 + A~D 选项） |
| `MaterialBlock` / `ShenlunQuestion` | 清洗 | 申论材料段落 / 作答要求 |
| `CleanedDocument` | 清洗输出 | 结构化文档 |
| `PageElement` / `LaidOutPage` / `LaidOutDocument` | 排版 | 带坐标的页面元素 / 排版结果 |
| `LayoutConfig` | 配置 | 20+ 可调排版参数 |

## 格式对照

| 刷题软件格式 | → | 考试标准格式 |
|------------|---|------------|
| 微软雅黑 / 黑体正文 | → | 宋体 (SimSun) 10.5pt |
| 仿宋正文（国考/联考预设） | → | 仿宋 (FangSong) 14pt / 12pt |
| 题号 `1.` `1、` `1。` | → | `1．` 全角点号（可配） |
| 选项 `A.` `A)` `A）` | → | `A．` 全角点号 |
| 非标准行距 | → | 可配倍数（默认 1.5） |
| 水印 / 广告 / 二维码 | → | 自动移除 |
| 无标准页眉 | → | `行政职业能力测验` / `申论` |
| 非标准纸张 | → | A4 / B5 / A3 / B4 / 16开 |

## 配置扩展

### 添加过滤规则

编辑 `config/xingce_filter_rules.json`，在 `regex_patterns` 中追加正则。匹配的文本行在清洗阶段被移除。

### 添加模块识别规则

编辑 `config/xingce_section_instructions.json`，在 `regex_patterns` 中追加正则。匹配的行被识别为模块标题。

### 自定义排版模板

编辑 `config/templates/xingce_template.json` 或 `shenlun_template.json`，所有参数有中文注释。

---

## 注意事项

- OCR 首次使用自动下载模型（约 200MB），需网络通畅
- 文本型 PDF 约 1~3 秒；扫描件速度取决于 OCR 后端，GPU 模式明显快于 CPU 模式
- 字体自动检测 `C:\Windows\Fonts`，缺失时自动回退
- 输出 PDF 嵌入字体子集，任意设备可正常显示

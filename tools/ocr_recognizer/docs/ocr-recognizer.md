# OCR 文字识别

从图片中提取文字，支持印刷体和手写体两种模式，识别结果可编辑、复制或导出为 TXT。

---

## 功能特性

- 📷 **图片识别**: 支持 PNG / JPG / BMP / TIFF / WebP / GIF 等常见图片格式
- ✍️ **手写体模式**: 专为手写文字优化的预处理和识别参数
- 🎨 **置信度标注**: 识别后图片上以颜色标注置信度（绿/黄/红）
- 📋 **一键复制**: 将识别文本复制到剪贴板
- 💾 **导出 TXT**: 保存为纯文本文件
- 🖱 **拖放支持**: 直接拖拽图片文件到窗口即可加载

---

## 使用说明

1. 点击 **「📂 打开图片」** 或拖放图片到左侧预览区
2. 选择文字类型：**印刷体**（默认）或 **手写体**
3. 点击 **「▶ 开始识别」**，等待 OCR 引擎处理
4. 识别完成后：
   - 左侧图片上会绘制文字区域边界框（颜色表示置信度）
   - 右侧显示识别出的纯文本
5. 点击 **「📋 复制」** 将文本复制到剪贴板，或 **「💾 导出 TXT」** 保存文件

## 识别管线

```
┌──────────┐    ┌──────────┐    ┌──────────┐
│  ① 加载   │ → │  ② 预处理  │ → │  ③ OCR    │
│  图片     │    │  增强     │    │  识别     │
└──────────┘    └──────────┘    └──────────┘
                     ↓               ↓
               ┌──────────┐    ┌──────────┐
               │  ⑤ 展示   │ ← │  ④ 标注   │
               │  结果     │    │  边界框   │
               └──────────┘    └──────────┘
```

### 预处理（手写体模式）

- 对比度增强（1.5×）
- 锐化（2.0×）
- 中值滤波去噪

印刷体模式仅做 RGB 格式归一化，保留原始图像质量。

### OCR 引擎

PaddleOCR 3.x + PaddlePaddle，CUDA 可用时自动启用 GPU 加速。

---

## 置信度颜色说明

| 颜色 | 置信度范围 | 含义 |
|------|-----------|------|
| 🟢 绿色 | ≥ 0.9 | 高置信度，识别结果可靠 |
| 🟡 黄色 | 0.7 – 0.9 | 中等置信度，建议人工复核 |
| 🔴 红色 | < 0.7 | 低置信度，可能需要手动修正 |

---

## 注意事项

- 首次使用 OCR 时需要下载模型文件（约 100 MB），请耐心等待
- 手写体模式对工整书写效果较好，潦草字迹识别率会下降
- 建议图片分辨率不低于 200 DPI 等效，过高分辨率会自动缩放到合理范围
- GIF 动图仅识别第一帧
- 如遇 OCR 依赖缺失，请运行项目根目录下的 `setup.bat` 安装

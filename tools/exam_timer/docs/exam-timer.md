# 考试计时器

简洁实用的倒计时 / 正计时工具，支持分段记录。

---

## 功能

- ⏱ **正计时**：从 0 开始往上计，适合记录做题总用时
- ⏲ **倒计时**：从设定时间倒数，到时后数字闪烁变红提醒
- 🏁 **分段记录**：计时过程中随时点击"分段"记录当前时间，右侧列表按序排列
- 🔢 **大字显示**：80px 粗体数字，一目了然

---

## 使用说明

1. 在首页点击 **「⏱ 考试计时器」** 进入
2. 选择计时模式：**正计时** / **倒计时**
3. 倒计时模式下设置目标时间（时:分:秒）
4. 点击 **「▶ 开始」** 开始计时
5. 需要记录分段点击 **「🏁 分段」**，当前时间会记录到右侧列表
6. 中途可 **「⏸ 暂停」** 再 **「▶ 继续」**
7. 点击 **「↺ 重置」** 归零并清空分段记录

### 倒计时提醒

- 剩余 ≤ 5 分钟时，数字自动变红提示
- 归零后数字红闪 + 标签显示"⏰ 时间到！"
